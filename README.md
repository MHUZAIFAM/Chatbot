# Chatbot Project

This repository contains two versions of a dataset analysis chatbot.

## Chatbot_V1.0_RuleBased
The original rule-based chatbot implementation.

Features:
- Rule-based query handling
- Dataset schema interpretation
- Section detection
- Ranking analysis

## Chatbot_V2.0_Agentic
An improved agentic chatbot architecture using deterministic tools and LLM reasoning.

Features:
- Agent-based architecture
- QueryEngine for deterministic dataset queries
- Gemini-powered reasoning
- Section-aware retrieval
- Conversation memory
- Streamlit interface
